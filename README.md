# Travel Itinerary Planner - Frontend

## 📝 Project Description
A full-stack travel planning application that helps users organize trips, track expenses, and discover destinations.

## ✨ Features
- User authentication (login/register)
- Browse destinations with category filters
- Create and manage trips
- Track expenses per trip
- Save destinations to wishlist
- Responsive design for all devices
- Dark/light mode support

## 🛠️ Tech Stack
- React 18
- Tailwind CSS
- ShadCN UI Components
- Framer Motion (animations)
- React Router DOM
- Axios for API calls
- Context API for state management
